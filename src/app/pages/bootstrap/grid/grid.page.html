Grid System

Bootstrap's responsive grid system is built on the CSS Flexible Box Layout module (flexbox) which allows up to 12 columns on a page. On a big screen, the 12 columns are organized side by side, but on a small screen, they will be stacked on top of each other.

If 12 individual columns are not needed, some columns can be grouped to create wider columns.

The Bootstrap 3 grid system has four classes based on screen sizes:

xs screens less than 768px wide (phones)

sm screens greater than or equal to 768px wide (tablets)

md screens greater than or equal to 992px wide (small laptops)

lg screens greater than or equal to 1200px wide (laptops and desktops)

To create a Bootstrap grid:

1. Create a container element, i.e. a <div> element with class container or container-fluid.

2. For each row, create a <div> element with class row.

3. For each column, create a <div> element with class col-*-# where * is the grid class and # is the number of columns to be used from the 12 available columns.

For example, the code below displays a simple Bootstrap grid with two rows having two differently sized columns each:

<div class="container">

<div class="row">

<div class="col-xs-6">6 cols</div>

<div class="col-xs-6">6 cols</div>

</div>

<div class="row">

<div class="col-xs-4">4 cols</div>

<div class="col-xs-8">8 cols</div>

</div>

</div>

The container class is for a responsive, fixed width container.

container-fluid is a full-width container.

While the container element is for wrapping site contents, rows are the containers for columns. The number of columns within each row should add up to 12.
