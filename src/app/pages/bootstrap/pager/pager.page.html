Pager

Pager refers to pagination in the form of previous and next buttons.

To create pager using Bootstrap, create a <ul> element with pager class. Next, add two <li> elements for Previous and Next buttons.

For Example:

<div class="container">

<ul class="pager">

<li<>a href="#">Previous</a<>/li>

<li<>a href="#">Next</a<>/li>

</ul>

</div>

To align the previous (next) buttons to the left (right) of the screen, use the previous (or next) class for the respective <li> element.

<div class="container">

<ul class="pager">

<li class="previous"<>a href="#">Previous</a<>/li>

<li class="next"<>a href="#">Next</a<>/li>

</ul>

</div>
