Affix

Bootstrap 3 provides an easy way to affix an element to an area on the page. An affixed element stays fixed at a specific area while scrolling the page.

To make an element affixed, just add the data-spy="affix" property.

The code below displays a div which stays at the same area on the screen while scrolling:

<div class="container">

<div id="affixed" data-spy="affix">

Affix

</div>
