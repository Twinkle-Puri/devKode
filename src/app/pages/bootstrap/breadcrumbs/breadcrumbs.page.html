Breadcrumbs

Breadcrumbs refers to pagination in the form of hierarchical navigation.

To create breadcrumbs using Bootstrap, create a <ul> element with breadcrumb class and then add one <li> element for each page.

For Example:

<div class="container">

<ul class= "breadcrumb">

<li<>a href="#" >Languages</a<>/li>

<li<>a href="#" >Programming</a<>/li>

<li<>a href="#" >C</a<>/li>

<li<>a href="#" >Intro</a<>/li>

</ul>

</div>
