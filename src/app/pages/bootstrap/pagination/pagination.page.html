Pagination

Bootstrap 3 provides an easy way to add pagination, i.e. page numbers, to a webpage.

To create pagination using Bootstrap, create a <ul> element with pagination class. Next, add one <li> element for each page.

The active/disabled classes can be used to highlight/disable page <li> elements.

For Example:

<div class="container">

<ul class="pagination" >

<li<>a href="#">1</a<>/li >

<li class="disabled"<>a href="#">2</a<>/li >

<li class="active"<>a href="#">3</a<>/li >

<li<>a href="#">4</a<>/li >

</ul >

</div >

Sizes of pagination blocks can be managed by using these classes.

pagination-sm Small blocks.

pagination-lg Large blocks.
