Wells

A well is a box with a gray background and rounded corners.

To create a well, use a <div> element with class well, i.e. <div class="well">

You can display wells of different sizes with these classes:

well-sm Small well.

well-lg Large well.

The code below displays wells of different sizes:

<div class="container">

<div class="well well-sm">Small well>/div>

<div class="well">Normal well</div>

<div class="well well-lg">Large well</div>

The well is generally used as a simple effect on an element to give it an inset effect.
