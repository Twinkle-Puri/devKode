Carousel

A carousel is used to cycle through content. It is also referred to as a slideshow because of the way it showcases videos and images.

The basic Bootstrap carousel has indicators (little circles) for the number of slides and the active slide and controls for manually moving backwards or forwards through a slideshow.

To build a carousel, start with a div with an id for referring to the carousel, the class="carousel", and attribute data-ride="carousel" for animating the carousel as soon as the page loads.

Within the outer div are the indicators, media content, and controls:

- The indicators are an ordered list with class="carousel-indicators". Each list item has a data-target attribute that points to the carousel id, and a data-slide-to attribute that indicates which slide to display when clicked.

- The media content is specified within a div with class carousel-inner.

- Within the carousel-inner div are separate div elements of class item for each media item. One item must also include the active class. Captions for each slide are placed in a div with class carousel-caption within the item's div.

- The controls are designed with <a> tags that have an href assigned the carousel id and a data-slide attribute set to "prev" or "next". Any character can be used to for next and prev controls, but Bootstrap comes with glyphs from the Glyphicon Halflings set. A glyphicon is inserted by using <span class="glyphicon glyphicon-glyphname>.

The code below displays a carousel:

<div class="container">

<div id="logoCarousel" class="carousel slide" data-ride="carousel">

<ol class="carousel-indicators black">

<li data-target="#logoCarousel" data-slide-to="0" class="active"></li>

<li data-target="#logoCarousel" data-slide-to="1"></li>

</ol>

<div class="carousel-inner">

<div class="item active">

<img src="https://images.freeimages.com/images/large-previews/9fc/yet-another-flower-1399208.jpg" style="width:100%">

<div class="carousel-caption">

<h3>Caption 1</h3>

</div>

</div>

<div class="item">

<img src="http://images.freeimages.com/images/large-previews/b3d/flowers-1375316.jpg" style="width:100%">

<div class="carousel-caption">

<h3>Caption 2</h3>

</div>

</div>

<a class="left carousel-control" href="#logoCarousel" data-slide="prev">

<span class="glyphicon glyphicon-chevron-left"<>/span>

</a>

<a class="right carousel-control" href="#logoCarousel" data-slide="next">

<span class="glyphicon glyphicon-chevron-right"<>/span>

</a>

</div>

</div>

Carousel is a plugin, and is included in the Bootstrap JS file.
