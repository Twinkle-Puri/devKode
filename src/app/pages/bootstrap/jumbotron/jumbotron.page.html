Jumbotron

A jumbotron is a big box which stands out from all other elements in a web page. It is displayed as a gray box with rounded corners.

To create a jumbotron, create a <div> element with class jumbotron , i.e. <div class="jumbotron">, as shown in the following code:

<div class= "container">

<div class="jumbotron">

<h1>DevKode</h1>

<p>Learn to code for FREE!</p>

</div>

</div>

A jumbotron generally enlarges the text inside it. If the jumbotron is placed outside the Bootstrap container, it extends to the edges of the screen.
