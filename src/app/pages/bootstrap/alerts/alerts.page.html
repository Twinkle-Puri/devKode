Alerts

Bootstrap provides an easy way to display alert messages. To create a Bootstrap alert, create a <div> element with class alert.

Alerts must be styled by one of the contextual classes: alert-info, alert-success, alert-warning, alert-danger.

To add a close (X) button to an alert, add an alert-dismissable class to the alert container. Then, add class="close" and data-dismiss="alert" to a link or button element inside the alert container.

The following code demonstrates different types of alerts provided by Bootstrap 3:

<div class="container">

<div class= "alert alert-success">

Success Alert!

</div>

<div class= "alert alert-warning">

Warning Alert!

</div>

<div class= "alert alert-danger">

Danger Alert!

</div>

<div class= "alert alert-info alert-dismissable">

<a class= "close" data-dismiss= "alert" href= "#">Close</a>

Information Alert!

</div>

</div>
