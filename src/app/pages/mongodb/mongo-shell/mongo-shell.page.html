MongoDB Shell

Here you can find the simple ways to traverse and view the documents of a MongoDB.

In your terminal, write mongo to start the Mongo environment:

$ mongo

Too see all the available databases type:

> show dbs

To select a database, enter:

> use databaseName

To see all the available collections:

> show collections

To see all the available documents (entries) of a certain collection:

> db.collectionName.find()

You can find the list of available operations on a collection here.
