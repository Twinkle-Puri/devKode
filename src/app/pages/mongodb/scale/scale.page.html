Vertical and Horizontal Scaling

In general, databases can be scaled with two different approaches, vertical and horizontal.

Vertical

In this method, you need to add more power to the existing server. For instance, you can add more CPU, RAM, and storage. This approach can be hard if you are maintaining your own server and not using cloud services like AWS.
